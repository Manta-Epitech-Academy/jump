# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** security@manta-music.com

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots

## Response Timeline

- **Acknowledgment:** within 48 hours
- **Assessment:** within 7 days
- **Fix (critical):** within 30 days

## Scope

This policy applies to the code in this repository. Infrastructure and deployment issues should be reported to the team directly.

## Supported Versions

Only the latest version on the `main` branch is supported with security updates.

# Story 4.3: Automated Parcoursup PDF Generation

**Status:** `Draft`
**Epic:** 4. Portfolio, Valorization, and Public Showcase

### User Story

**As a** Student (or Parent),
**I want** to download a formal certificate summarizing my workshop,
**so that** I can add it to my academic files (like Parcoursup).

### Acceptance Criteria

1.  **UI Integration:** A "Télécharger mon Attestation" button is added to the student's main dashboard (`/(camper)/camper/+page.svelte`), visible if they have completed at least one event.
2.  **PDF Generation:** The button triggers a backend endpoint (similar to the existing staff diploma generator).
3.  **Content:** The PDF includes the student's name, the event title(s), total XP gained, and the official Epitech branding.
